# SNSX ExOS Roadmap

## Phase 1

- physical memory manager
- heap allocator
- timer foundation
- ATA boot-disk driver
- read-only packaged app volume
- ELF program loader

Status:
- implemented in the current repo baseline, with the next step being stronger isolation and a fuller process model

## Phase 2

- process model
- syscall interface
- protected user-space runtime
- package and app format
- persistent configuration

## Phase 3

- PCI enumeration
- Ethernet support
- IPv4, UDP, TCP, sockets
- DNS and service discovery
- secure update channel

## Phase 4

- USB host stack
- Bluetooth controller support
- Wi-Fi chipset support
- wireless authentication
- network manager UI

## Phase 5

- graphics mode driver
- compositor and window system
- audio stack
- native apps
- education-platform port into `SNSX ExOS`
